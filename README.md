# 洛克王国世界全自动辅助

作者：勼欢Ryuu

这是一个面向单人高效率场景、支持长时间挂机的洛克王国奇丽花脚本，基于 AutoHotkey v2 编写，内置动作表情循环、白天倒计时与自动流程衔接等功能。

核心目标是尽量解放双手，让单人挂机更省心、更稳定；竞技场 PVP 自爆功能正在开发中。

当前版本重点放在这几件事：

- 提供可视化控制面板与设置窗口
- 支持自定义启动热键、动作表情键和白天倒计时
- 首次使用带说明页，运行设置可自动保存
- 可直接运行打包后的 `LuoKeAuto.exe`

如果你只是想快速开始：

1. 在 GitHub Release 下载 `LuoKeAuto.exe`。
2. 必须以管理员身份运行。
3. 传送到目标位置，调整时间为早上。
4. 打开设置确认热键与倒计时。
5. 回到游戏前台，按启动热键开始运行。

## 功能概览

- 面向单人高效率奇丽花挂机场景，适合长时间运行
- 前台自动执行动作表情、生产、收取与确认流程
- 支持白天倒计时，到点后自动执行一轮收放精灵流程
- 提供深色控制面板、设置窗口和首次使用说明页
- 首次保存设置后会在本地生成 `LuoKe.ini`，用于保存热键和倒计时；该文件默认不纳入仓库

## 环境要求

- Windows
- 游戏窗口标题可匹配“洛克王国”

运行源码时还需要：

- AutoHotkey v2

运行打包后的 EXE 时：

- 无需额外安装 AutoHotkey

## 使用方式

1. 先将角色传送到准备挂机的位置。
2. 调整游戏内时间为早上。
3. 必须先以管理员身份运行脚本。
4. 打开脚本设置，确认热键、动作表情键和倒计时。
5. 回到游戏前台，按设置好的启动热键开始运行。

## 默认配置

- 启动/停止热键：`PgDn`
- 动作表情按键：`P`
- 白天倒计时：`1` 分钟

说明：

- 启动/停止热键支持 `Ctrl/Alt/Shift/Win + 其他键`
- 动作表情按键仅支持单键
- 脚本内已内置首运行默认值；如果本地存在 `LuoKe.ini`，则优先读取该文件中的个人设置

## 运行源码

1. 安装 AutoHotkey v2。
2. 运行 `LuoKe.ahk`。
3. 在游戏前台按设置好的启动热键开始或停止。

## 直接运行 EXE

1. 双击运行 `LuoKeAuto.exe`。
2. 在控制面板中确认设置。
3. 回到游戏前台按启动热键运行。

## 注意事项

- 当前脚本为前台执行模式，运行期间不要让游戏窗口长时间失焦。
- 必须以管理员身份运行，否则部分按键发送和前台交互可能不稳定。
- 如果你修改过游戏里的动作表情键，请在脚本设置里同步修改。
- 建议先设置 `1` 分钟循环，确认流程稳定后再提高挂机时长。
- 首次使用前请先阅读程序内的使用说明与风险提示。
- `LuoKe.ini` 是运行时自动生成的本地配置文件，建议保留在本机，不要提交到 GitHub。

## 构建 EXE

如果你已经安装 AutoHotkey 编译器 `Ahk2Exe.exe`，可在项目目录执行：

```powershell
.\build.ps1
```

构建完成后，会在 `dist\release\` 下生成发布目录，包含：

- `LuoKeAuto.exe`
- `README.md`
- `LICENSE`
